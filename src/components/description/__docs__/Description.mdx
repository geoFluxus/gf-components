# Description

Base component for titles and subtitles.

### Two columns (default)

<Canvas of={Description.TwoColumns} />

### Three columns 

<Canvas of={Description.ThreeColumns} />

## Usage

```ts
import {Description} from "gf-components";

const Example = () => {
  return (
    <Description
      text={text}
      columns={columns}
    />
  );
};

export default Example;
```

#### Arguments
- **text** - An array of strings that represents the body of the Description.
- **columns** - The number of columns, either 2 | 3 | 4.
