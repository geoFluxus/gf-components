# Description

Base component for titles and subtitles.

### Two columns (default)

<Canvas of={Description.TwoColumns} />

### Two columns (editable)

<Canvas of={Description.TwoColumnsEditable} />

### Three columns 

<Canvas of={Description.ThreeColumns} />

## Usage

```ts
import {Description} from "gf-components";

const Example = () => {
  return (
    <Description
      text={text}
      columns={columns}
      editable={editable}
    />
  );
};

export default Example;
```

#### Arguments
- **text** - A string that represents the body of the Description.
- **columns** - The number of columns, either 2 | 3 | 4.
- **editable** - Boolean that makes the description editable (for translation purposes).
