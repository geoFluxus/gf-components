# GFButton

Ant Design Button component with styling adapted, which can have icons or not.

### Example (primary button)

<Canvas of={GFButton.Primary} />

## Usage

```ts
import {GFButton} from "gf-components";

const Example = () => {
  return (
      <GFButton
        disabled={false}
        label={label}
        onClick={()=> alert("Clicked")}
        type={'default'}
      />
  );
};

export default Example;
```

#### Arguments

- **label** - ReactNode that represents the label of the button.
- **type** - A string with one of four possible values: "link", "text", "primary", "default" or "dashed" . Typically, a primary button stands out as the main action in a user interface.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **onClick** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.


### Example of Buttons with Icons (download button)

Icons can be added to the button via the 'icon' property, with is position defined by the 'iconPosition' property ('start'|'end'). Please refer back to Ant Design Button documentation for additional information.

<Canvas of={GFButton.Download} />

## Usage

```ts
import {GFButton} from "gf-components";
import { DownloadOutlined } from '@ant-design/icons';

const Example = () => {
  return (
      <GFButton
        disabled={false}
        label={'Download}
        type={'default'}
        icon={<DownloadOutlined />}
        iconPosition={'start'}
        onClick={()=> alert("Clicked")}
      />
  );
};

export default Example;
```

#### Arguments

- **label** - ReactNode that represents the label of the button.
- **type** - A string with one of four possible values: "link", "text", "primary", "default" or "dashed" . Typically, a primary button stands out as the main action in a user interface.
- **icon** - A React.ReactNode element with the icon.
- **iconPosition** - A string with either "start" or "end" values.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **onClick** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
