# Metrics

Metrics to be used as base for the remaining components.

### Example (default)

<Canvas of={Metrics.Default} />

## Usage

```ts
import {Metrics} from "gf-components";

const Example = () => {
  return (
    <Metrics
      percent={35},
      cost={18.6},
      description={'description of the metrics'},
      editable={false},
    />
  );
};

export default Example;
```

### Editable Metric card

<Canvas of={Metrics.Editable} />

## Usage

```ts
import {Metrics} from "gf-components";

const Example = () => {
  return (
    <Metrics
      percent={35},
      cost={18.6},
      description={'description of the metrics'},
      editable={true},
    />
  );
};

export default Example;
```

#### Arguments
- **percent** - The percentual value of the metrics.
- **cost** - The cost of the metrics, in euros.
- **description** - The content of the card.
- **editable** - Boolean that makes the description of the Metric editable (for translation purposes).
