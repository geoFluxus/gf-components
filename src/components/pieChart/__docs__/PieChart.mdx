# PieChart

Base component for Pie chart.

### Example

<Canvas of={PieChart.Default} />

## Usage

```ts
import {PieChart} from "gf-components";

const Example = () => {
  return (
    <PieChart
      pieChartData={data}
      title={title}
    />
  );
};

export default Example;
```

#### Arguments
- **PieChartData** - Object containing nodes for Pie chart
- **title** - Title for the pie chart
