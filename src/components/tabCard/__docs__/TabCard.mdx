# TabCard

Card with tabs and optional expandable section.

#### Example

<Canvas of={TabCard.Default} />

## Usage

```ts
import {TabCard} from "gf-components";

const Example = () => {
  return (
    <TabCard 
      tabList={tabList}                   // List of tabs
      tabContent={tabContent}             // Object with content for each tab 
      expandContent={expandContent}       // Object with the content do be displayed in the expandable for each tab (optional)
      expandLabel={expandLabel}           // Label for the expandable button (optional)
      handleDownload={handleDownload}     // Handler for the download button
      {...props}                          // Additional props (optional -- see GFCard / AntD Card)
    />
  );
};

export default Example;
```

#### Arguments
- **tabList** - Array of objects of the form ```{ key: string, label: string }``` corresponding to the tabs of the card.
- **tabContent** - Object that holds the content of each tab, where each key corresponds to the key of the tab (see tabList above) and the content can be of type React.ReactElement | React.ReactNode.
- **expandContent** - Object that holds the content of the expandable section of each tab, where each key corresponds to the key of the tab (see tabList above) and the content can be of type React.ReactElement | React.ReactNode.
- **expandLabel** - String for the label of the expandable button.
- **handleDownload** - Function that is triggered when someone clicks the download button.
