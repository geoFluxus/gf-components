# Trends

Base component for Trends chart.

### Example

<Canvas of={Trends.Default} />

## Usage

```ts
import {Trends} from "gf-components";

const Example = () => {
  return (
    <Trends
    trendsData={data}
    tooltip={tooltip}
    />
  );
};

export default Example;
```

#### Arguments
- **trendsData** - Object containing nodes for Trends
- **tooltip** - Function returning JSX element as custom tooltip
