# VisSelectors

Select dropdown component for visualisation.

#### Example

<Canvas of={VisSelectors.Default} />

## Usage

```ts
import {VisSelectors} from "gf-components";

const Example = () => {
  return (
    <VisSelectors isOneSelector={value}/>
  );
};

export default Example;
```

#### Arguments
- **isOneSelector** - Boolean value to determine if one or two selectors are rendered
