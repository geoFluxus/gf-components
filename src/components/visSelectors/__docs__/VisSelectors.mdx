# VisSelectors

Select dropdown component for visualisation.

#### Example

<Canvas of={VisSelectors.Default} />

## Usage

```ts
import {VisSelectors} from "gf-components";

const Example = () => {
  return (
    <VisSelectors selectors={selectors}/>
  );
};

export default Example;
```

#### Arguments
- **selectors (Array)** - Array of selector objects

    *Selector object*
    - id (string) - To refer to the selector in the state setter
    - title (string) - Selector title
    - defaultValue (string) - Selector option object value to use as default
    - options (Array) - Array of selector option objects

        *Selector option object*
        - label (string): Text to show for given selector option
        - value (string): Key to recover for given selector option

- **setter (function)** - Function to update parent state with args
    - key (string): Selector key to update
    - value (Array): Array of values to update selector for given id
